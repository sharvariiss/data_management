# Week 1 - Metadata Exercise

## Dataset
New York City Airbnb Open Data

## Source
Kaggle: https://www.kaggle.com/datasets/dgomonov/new-york-city-airbnb-open-data

## Description
This exercise creates metadata for the New York City Airbnb Open Data dataset.  
The dataset contains Airbnb listing information for New York City in 2019, including location, neighbourhood, room type, price, minimum nights, reviews, and availability.

## Metadata Files
- `nyc_airbnb_datacite.xml`
- `nyc_airbnb_schemaorg.json`
